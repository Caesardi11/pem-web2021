Tutorial Semantik Element untuk Pemula

Semantik Element adalah elemen yang memiliki makna dan tujuan. Tujuannya agar kode HTML mudah dibaca dan tidak ada penyalahgunaan tag. Element Semantik bagus untuk SEO dan juga dapat meningkatkan accessibility.
